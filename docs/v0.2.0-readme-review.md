## What needs human review before commit

Nothing blocks commit. But scan these manually if you have time:

1. **Quickstart command (lines ~38–42)** — one-line chain `init && run && report --html`. Verify it reads well for someone with 2 kb working memory.
2. **"Your first run will show 0%" section tone** — does it reassure without sounding defensive?
3. **Hero screenshot** (now `screenshot-05-actions.png`) — is this the right demo to lead with, or should the position grid (02) be lead?
4. **Price trio rounding** — I used $0.20 / $0.50 / $0.55. Real numbers from live runs were $0.17–$0.21 / $0.48–$0.52 / $0.53–$0.57. Rounding is defensible; flag if you'd prefer ranges ($0.18–$0.22 / $0.45–$0.55).
